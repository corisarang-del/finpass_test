## 공통 추가 규칙 (2026-02-13)

### 홈페이지 일관성 우선 규칙
- 기존 홈페이지가 이미 시각 기준을 갖고 있으면, 신규/개편 페이지도 동일한 톤을 우선 적용한다.
- 최소 일관성 항목:
  - 폰트: 홈에서 쓰는 폰트 스택 유지
  - 배경: 홈과 같은 계열(딥 네이비/라이트 블루 톤) 유지
  - 버튼: 라운드/강조 색상/호버 대비 규칙 유지
  - 레이아웃: 카드 반경, 여백, 그림자 강도, 섹션 폭 일관성 유지
- 구현 순서:
  1. `Home.tsx`와 공통 스타일(`index.css`, `App.css`) 먼저 확인
  2. 색/타이포/버튼 토큰을 기존 화면과 맞춤
  3. 신규 화면은 기존 흐름을 끊지 않게 라우팅 연결
- 금지:
  - 기존 톤과 무관한 임의 컬러 테마 도입
  - 페이지마다 다른 버튼 스타일/간격 체계 혼용

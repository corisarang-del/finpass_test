## 수행한 작업

### 1. Home.tsx 수정
#### 인트로 텍스트 중앙 정렬 강화 (67-88줄)
- 고정 위치 컨테이너에 명시적 스타일 추가: `style={{ top: 0, left: 0, right: 0, bottom: 0 }}`

- 텍스트 요소에 `text-center` 명시
- h1 drop-shadow를 `drop-shadow-2xl`로 강화

#### 카테고리 레이아웃 및 호버 효과 개선 (100-143줄)
- 최소 높이 추가: `min-h-[200px]` (모바일)
- 이미지 opacity 조정: `opacity-70` → 호버 시 `opacity-100`
- 이미지 scale 효과 감소: `scale-110` → `scale-105`
- 오버레이를 그라데이션으로 변경: `bg-gradient-to-b from-black/40 via-black/50 to-black/60`
- 배지 항상 표시, 호버 시 border와 opacity 변경
- Description을 카드 내부로 통합하여 레이아웃 단순화
- 호버 시 scale 효과 추가: `group-hover:scale-105`
- 타이틀 크기 증가: `text-4xl md:text-5xl lg:text-6xl`

- 사용하지 않는 `ArrowRight` import 제거

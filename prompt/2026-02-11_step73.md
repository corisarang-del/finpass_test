## 수행한 작업

### 1. 근본 원인 분석
- CSS Grid + `motion.button` 조합이 제대로 작동하지 않음
- button 요소의 기본 스타일이 grid cell을 채우지 못함
- Tailwind CSS 클래스가 framer-motion과 충돌 가능

### 2. Home.tsx 카테고리 섹션 전면 재작성 (100-154줄)

#### 컨테이너: Grid → Flex + 인라인 스타일
```tsx
style={{
  position: 'fixed',
  top: 0, left: 0,
  width: '100vw',
  height: '100vh',
  display: 'flex',
  flexDirection: 'column',
  overflow: 'hidden',
}}
```

#### 카테고리 카드: motion.button → motion.div + 인라인 스타일
```tsx
style={{
  position: 'relative',
  width: '100%',
  height: '25vh',
  overflow: 'hidden',
  cursor: 'pointer',
  margin: 0, padding: 0, border: 'none',
}}
```

#### 이미지: Tailwind 클래스 → 인라인 스타일
```tsx
style={{
  position: 'absolute',
  top: 0, left: 0,
  width: '100%', height: '100%',
  objectFit: 'cover',
  display: 'block',
}}
```

## 기술적 세부사항

### Vite의 이미지 처리
1. **Import 방식**: ESM import로 이미지를 가져오면 Vite가 자동 처리
2. **빌드 최적화**: 이미지를 assets 폴더에 복사하고 hash 추가
3. **타입 안정성**: TypeScript가 import 경로를 체크
4. **트리 쉐이킹**: 사용되지 않는 이미지는 번들에서 제외

### 성능 개선
- **로컬 파일**: 네트워크 요청 없이 즉시 로드
- **캐싱**: 브라우저가 이미지를 캐싱하여 재방문 시 빠른 로드
- **오프라인**: 인터넷 연결 없이도 작동

### PS 로고 문제 해결
- Unsplash URL 제거로 외부 리소스 의존성 완전 제거
- 로컬 이미지만 사용하여 안정성 향상

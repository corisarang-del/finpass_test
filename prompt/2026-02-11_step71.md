## 기술적 세부사항

### Tailwind CSS Preflight 문제
- Tailwind의 `@tailwind base`는 Preflight라는 normalize.css 기반 리셋 포함
- 기본 margin, padding을 제거하지만 브라우저마다 다르게 적용될 수 있음
- 명시적으로 html, body 스타일을 Tailwind 로드 후 재정의

### 완전한 Fullscreen을 위한 설정
1. **html**: 100% 크기, overflow hidden
2. **body**: 100% 크기, overflow hidden
3. **#root**: 100vh 크기, overflow hidden
4. **Home 컴포넌트**: fixed inset-0

### Overflow Hidden의 역할
- 의도하지 않은 스크롤바 제거
- 모바일에서 주소창 등으로 인한 높이 변화 시에도 스크롤 방지
- 완벽한 fullscreen 경험 보장

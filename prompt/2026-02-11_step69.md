## 기술적 세부사항

### Flex vs Grid 비교
- **Flex 문제점**: 
  - content 크기나 padding에 따라 각 아이템의 크기가 미묘하게 달라질 수 있음
  - min-height 사용해도 여백이 생길 수 있음
  
- **Grid 장점**:
  - 명시적으로 4등분 보장
  - 각 셀이 정확히 동일한 크기
  - `grid-rows-4` (모바일 세로 4개) / `grid-cols-4` (데스크탑 가로 4개)
  - `h-full`로 각 버튼이 셀 전체를 채움

### 이미지 풀사이즈 원리
- Grid cell이 화면을 정확히 4등분
- 버튼이 `w-full h-full`로 cell 전체를 채움
- 이미지가 `absolute inset-0`로 버튼 전체를 채움
- 결과: 여백 없이 풀사이즈 이미지 표시

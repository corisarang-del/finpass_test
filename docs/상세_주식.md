# 상세 흐름 - 주식 성장 코스 (트레이더 최)

## 1. 시작 상황
- 투자에 관심은 있지만 변동성에 불안을 느끼는 사용자

## 2. 주요 선택지 예시
- 투자 경험: 없음 / 소액 / 정기 투자
- 목표 기간: 3년 / 5년 / 10년 이상
- 위험 성향: 낮음 / 중간 / 높음

## 3. 카테고리 추가 질문 예시
- 손실이 발생했을 때 감내할 수 있는 수준은 어느 정도인가요?
- 주식 외에 다른 성장 자산을 고려하고 계신가요?
- 투자 원칙을 한 문장으로 말한다면 무엇인가요?

## 4. 가이드 등장
- “이번 코스를 담당하게 된 수석 가이드 트레이더 최입니다.”

## 5. 가이드 피드백(예시 대사)

## 6. 최종 분석 포인트

# 개발문서 - 핀패스(FinPass)

## 1. 기술 스택(기준)
- Vite + React
- 스타일: CSS 모듈 또는 CSS-in-JS 중 선택 가능
- 패키지 매니저: pnpm

## 2. 화면 구조(IA)
- / (홈)
- /wizard (질문 흐름)
- /result (이정표 결과)
- /course/:type (상세 흐름 페이지)

## 3. 주요 모듈
- QuestionBank: 공통 질문 + 카테고리 추가 질문
- GuideProfiles: 가이드 캐릭터 데이터
- MilestoneEngine: 이정표 생성 로직(룰 기반)
- FinalAnalysis: 최종 분석 생성
- TimelineView: 결과 타임라인 렌더

## 4. 상태 흐름
- 홈 진입 → 4개 카테고리 선택
- 가이드 인사 후 질문 진행
- 질문 단계 입력 수집 → 결과 생성(인생 지도)
- 최종 분석 → 상세 흐름 또는 액션 브리지

## 5. 데이터 구조(초안)
- 질문
  - id, 단계, 질문문, 선택지, 후속 분기, 카테고리 태그, 가이드 멘트
- 가이드
  - id, 이름, 톤, 인사 멘트, 카테고리, 시각 키워드
- 결과
  - 이벤트 유형, 예상 시점, 비용 범위, 메시지
- 최종 분석
  - 현재 단계, 주요 리스크, 핵심 기회, 추천 액션

## 6. UX 원칙
- 공감형 톤으로 불안을 완화
- 숫자/전문용어는 필요 시점에만 노출
- 가이드 등장은 “선택 후 등장” 규칙 고정
- 질문은 한 화면 한 질문 중심(몰입도 유지)

## 7. 향후 확장
- 사용자 계정/저장
- 6개월 리마인더 메일링
- What-if 시뮬레이션 모듈

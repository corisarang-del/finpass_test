# 흐름 문서 - 핀패스

## 1. 홈(랜딩)
- 레퍼런스 기반 카피 3종을 순환 또는 A/B 테스트
  - “당신의 삶을 데이터로 읽다”
  - “보이지 않던 길이 열리다”
  - “꿈꾸던 미래를 현실의 전략으로”
- 상단 태그(칩): DATA INTELLIGENCE / STRATEGIC ROADMAP / FUTURE REALITY
- 4개 카테고리 카드 노출

## 2. 카테고리 선택
- 부동산 / 기초자산(보험·연금) / 주식 / 라이프밸런스
- 선택 직후 담당 가이드가 등장해 인사
- 가이드가 질문 흐름을 “직접 제시”하는 연출

## 3. 질문 흐름(Wizard)
- 발견 → 관계 → 공간 → 자유 → 현실 연결 순서
- 화면 1개 = 질문 1개 입력(레퍼런스 이미지 스타일)
- 공통 질문 후, 카테고리 추가 질문이 중간에 삽입됨
- 상세 UI는 `docs/질문UI스펙.md` 참고

## 4. 결과(인생 지도) 및 최종 분석
- 타임라인으로 주요 이벤트 표시
- 이벤트 중첩은 “충돌 가능성 안내”로 표현
- 현재 단계 진단, 리스크/기회 요약, 다음 행동 제안
- 상세 구성은 `docs/최종분석화면.md` 참고
- 미션 주기는 `docs/미션주기.md` 참고

## 5. 상세 흐름
- 선택지 제공 → 사용자의 선택 → 가이드의 피드백/설명
- 결과 요약 + 다음 행동 제안(CTA)

## 6. 액션 브리지
- 목표 역산 또는 재무 코치 연결
- 6개월 리마인드 설정 제안

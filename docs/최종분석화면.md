# 최종 분석 화면 구성 - 핀패스

## 목적
- 질문 답변을 기반으로 “현재 단계”와 “다음 행동”을 한 화면에서 정리한다
- 불안을 자극하지 않고, 실행 가능한 선택지를 제시한다

## 화면 구조
- 상단 헤더
- 핵심 요약 카드
- 인생 지도(타임라인)
- 리스크/기회 요약
- 다음 행동(CTA)
- 코스 재선택/공유
- 미션 주기 안내

## 1. 상단 헤더
- 타이틀: “OOO님의 인생 지도”
- 서브: “오늘의 선택이 1년 뒤의 나를 만듭니다”
- 배지: 선택한 카테고리 + 가이드 이름

## 2. 핵심 요약 카드(카드별 카피/수치 예시)

### 카드 A: 현재 단계 진단
- 라벨: 씨앗 / 성장기 / 정체 / 혼란
- 예시 카피: “지금은 씨앗 단계입니다. 목표까지의 시간이 충분하고, 작은 습관 변화가 큰 영향을 줍니다.”

### 카드 B: 준비 지수(정성+수치 혼합)
- 지수 범위: 0~100
- 상태 배지: 안정 / 주의 / 경고
- 예시: “준비 지수 62 (주의)”
- 설명: “방어 자산이 부족해 변수에 취약합니다.”

### 카드 C: 목표 시점 전망
- 예시 수치: “목표까지 5년 4개월”
- 보조 카피: “현재 속도 유지 시 +8개월 지연 가능”

### 카드 D: 월간 여력 요약
- 예시 수치: “가용 저축 35만 원 / 추천 45만 원”
- 보조 카피: “월 10만 원 조정 시 목표 시점 6개월 단축”

### 카드 E: 이벤트 충돌 체크
- 예시: “2029년 주거 + 출산 일정 중첩 가능성”
- 보조 카피: “우선순위 조정이 필요합니다.”

### 카드 F: 한 줄 요약
- 예시: “지금은 방어력을 올리는 것이 가장 큰 지름길입니다.”

## 3. 인생 지도(타임라인)
- 연도 기준 주요 이벤트 표시
- 이벤트 아이콘: 결혼, 주거, 출산, 은퇴, 여행/연수 등
- 중첩 이벤트는 “충돌 가능성 안내”로 표시
- 클릭 시 간단한 설명 카드 노출

## 4. 리스크/기회 요약
- 리스크 Top 2
  - 예: “대출 비중이 높아 금리 변동에 취약”
  - 예: “비상금 부족으로 갑작스러운 변수에 약함”
- 기회 Top 2
  - 예: “지출 구조가 유연해 조정 여지가 큼”
  - 예: “목표 시점이 길어 투자 전략 확장 가능”

## 5. 다음 행동(CTA)
- 1차 CTA: “목표 역산 플랜 시작하기”
- 2차 CTA: “월간 액션 체크리스트 받기”
- 보조 CTA: “가이드와 한 번 더 설계하기”

## 6. 코스 재선택/공유
- 다른 코스 탐색 버튼
- 결과 요약 공유(이미지 카드 형태)

## 7. 미션 주기 안내
- 주간: “이번 주 1가지 습관 점검”
- 월간: “한 달 성과 리포트 + 다음 달 액션 제안”
- 분기: “리스크/목표 재정렬 점검”
- 6개월: “중장기 경로 재설계 리마인드”
- 상세 주기는 `docs/미션주기.md` 참고

## 8. 카테고리별 카드 카피 예시

### 부동산(에이전트 한)
- 현재 단계: “주거 목표가 뚜렷해졌습니다. 이제 시점과 예산을 좁히면 됩니다.”
- 준비 지수: “준비 지수 58 (주의) — 대출 리스크가 핵심 변수입니다.”
- 목표 시점: “목표까지 6년 2개월 — 청약 전략 시 10개월 단축 가능”
- 월간 여력: “가용 저축 40만 원 / 추천 55만 원”
- 이벤트 충돌: “2030년 주거 + 결혼 일정 중첩 가능성”
- 한 줄 요약: “지금은 ‘입지 확정’보다 ‘자금 구조 정리’가 먼저입니다.”

### 기초자산(가디언 송)
- 현재 단계: “방어 기초가 부족합니다. 우선 비상금부터 세우세요.”
- 준비 지수: “준비 지수 66 (주의) — 비상금과 보험 목적 재정비 필요”
- 목표 시점: “안정 단계까지 18개월 — 월 10만 원 조정 시 6개월 단축”
- 월간 여력: “가용 저축 30만 원 / 추천 40만 원”
- 이벤트 충돌: “2028년 출산 + 보험 갱신 시기 중첩 가능성”
- 한 줄 요약: “지금은 속도보다 안정이 우선입니다.”

### 주식(트레이더 최)
- 현재 단계: “투자 원칙이 아직 불안정합니다. 기간을 먼저 정하세요.”
- 준비 지수: “준비 지수 70 (주의) — 변동성 허용 범위 재정의 필요”
- 목표 시점: “목표까지 7년 — 분산 투자 시 변동 리스크 20% 감소”
- 월간 여력: “가용 투자 25만 원 / 추천 35만 원”
- 이벤트 충돌: “2029년 주거 자금과 투자 납입 충돌 가능성”
- 한 줄 요약: “원칙이 정리되면 수익은 따라옵니다.”

### 라이프밸런스(디렉터 유)
- 현재 단계: “삶의 목표가 선명해졌습니다. 이제 현실적 조정만 남았습니다.”
- 준비 지수: “준비 지수 64 (주의) — 소비 조정 여지가 큽니다.”
- 목표 시점: “목표까지 3년 6개월 — 지출 10% 조정 시 5개월 단축”
- 월간 여력: “가용 저축 20만 원 / 추천 35만 원”
- 이벤트 충돌: “2027년 여행 + 결혼 비용 일정 중첩 가능성”
- 한 줄 요약: “지금의 선택이 더 나다운 미래를 만듭니다.”

## 카피 톤 원칙
- 공감형 문장 + 실행 유도 문장 조합
- 불안 조장 대신 “지금부터 바꾸면 된다”는 메시지 강조

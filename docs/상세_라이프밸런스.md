# 상세 흐름 - 라이프 밸런스 코스 (디렉터 유)

## 1. 시작 상황
- 결혼/여행/연수 등 삶의 목표를 재무와 연결하고 싶은 사용자

## 2. 주요 선택지 예시
- 가장 중요한 삶의 목표: 결혼 / 여행 / 자기계발
- 희망 시점: 1~2년 / 3~5년 / 5년 이상
- 우선 가치: 자유 / 안정 / 성장

## 3. 카테고리 추가 질문 예시
- 목표 달성 후 어떤 장면을 가장 먼저 그리고 싶나요?
- 현재 소비 중 줄일 수 있는 항목은 무엇인가요?
- 목표를 위해 포기할 수 없는 가치는 무엇인가요?

## 4. 가이드 등장
- “이번 코스를 담당하게 된 수석 가이드 디렉터 유입니다.”

## 5. 가이드 피드백(예시 대사)
- “돈을 모으는 것만큼 중요한 건, 그 돈으로 어떤 기억을 살 것인가입니다.”

## 6. 최종 분석 포인트
- 목표 일정의 현실성
- 삶의 가치와 재무 균형

## 7. 결과 요약
- 목표와 재무 연결 제안 + 현실적 조정 포인트

# 홈 애니메이션 콘셉트 - 핀패스

## 목적
- 랜딩 첫 화면에서 “데이터로 삶을 읽는 서비스” 인상을 강하게 전달
- 감정 톤: 차분함 + 신뢰감 + 기대감

## 전체 시퀀스(15~20초 루프)
1) 어두운 배경에 미세한 별/입자 등장
2) 지평선 곡선(지도/지구 곡면) 라인이 서서히 그려짐
3) 텍스트 1 등장: “당신의 삶을 데이터로 읽다”
4) 텍스트가 흐려지며 태그 칩 등장: DATA INTELLIGENCE
5) 다음 장면 전환(배경이 흐릿한 손/업무 장면)
6) 텍스트 2 등장: “보이지 않던 길이 열리다”
7) 태그 칩: STRATEGIC ROADMAP
8) 마지막 장면 전환(따뜻한 하늘/미래 이미지)
9) 텍스트 3 등장: “꿈꾸던 미래를 현실의 전략으로”
10) 태그 칩: FUTURE REALITY
11) CTA 버튼 강조(살짝 밝아짐/확대)
12) 1~2초 멈춤 후 1번으로 자연스럽게 루프

## 화면 구성 요소
- 상단 좌측 로고: FinPass
- 상단 우측 버튼: 시작하기
- 중앙 타이틀: 대형 한글 카피
- 보조 문구: 작은 서브 카피 1~2줄
- 태그 칩: 영어 키워드 1개씩 표시

## 인터랙션
- 마우스 이동 시 배경에 아주 미세한 패럴랙스
- CTA 버튼 hover 시 그림자 + 밝기 상승
- 스크롤 시작 시 애니메이션 페이드아웃, 카테고리 카드 등장

## 카피 예시(홈 카피 3종)
1) “당신의 삶을 데이터로 읽다”
   - “FinPass가 당신만의 지도 를 그립니다.”
2) “보이지 않던 길이 열리다”
   - “목적지까지 가는 최적의 경로.”
3) “꿈꾸던 미래를 현실의 전략으로”
   - “막연한 불안 대신, 명확한 이정표를 세우세요.”

## 카테고리 카드 진입 연출
- 애니메이션 종료 지점에 카드가 아래에서 위로 슬라이드 인
- 4개 카드가 0.1~0.2초 간격으로 순차 등장
- 카드 내용: 아이콘 + 타이틀 + 한 줄 설명

## 카드 카피 예시
- 부동산: “내 집 마련의 현실을 계산하다”
- 기초자산: “흔들리지 않는 방어를 설계하다”
- 주식: “성장의 속도를 숫자로 조정하다”
- 라이프밸런스: “삶의 목표를 돈과 연결하다”

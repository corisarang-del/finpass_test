# 미션 주기 - 핀패스

## 목적
- 단발성 분석에 그치지 않도록 “작은 실행”을 주기로 연결한다
- 부담 없이 반복 가능한 루틴을 만든다

## 주기 구성

### 주간(Weekly)
- 주간 미션 1개 제시
- 예시 카피
  - “이번 주, 고정지출 1개만 줄여보세요.”
  - “이번 주, 지출 내역을 1번만 기록해보세요.”

### 월간(Monthly)
- 월간 리포트 + 다음 달 액션 제안
- 예시 카피
  - “이번 달 목표 달성률 72%입니다.”
  - “다음 달에는 식비 10% 조정이 효과적입니다.”

### 분기(Quarterly)
- 목표/리스크 재점검
- 예시 카피
  - “지난 분기와 비교해 목표 시점이 3개월 앞당겨졌습니다.”
  - “금리/지출 변화에 따라 경로를 다시 계산합니다.”

### 6개월(Half-year)
- 중장기 경로 재설계 리마인드
- 예시 카피
  - “6개월 전의 계획을 다시 점검할 시간입니다.”
  - “지금의 속도로는 목표 시점이 4개월 늦어질 수 있습니다.”

## 푸시/메일 톤 원칙
- 압박 금지, “함께 설계” 느낌 강조
- 숫자와 감정 메시지를 동시에 제공

## 보상 연출(선택)
- 미션 완료 시 작은 보상 카드
- 예시: “이번 달, 미래로 1칸 전진했습니다.”

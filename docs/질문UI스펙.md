# 질문 화면 UI 스펙 - 핀패스

## 1. 기본 레이아웃
- 상단: STEP 표시 + 단계명(발견/관계/공간/자유/연결)
- 중앙: 질문 텍스트
- 입력 영역: 질문 유형에 맞는 UI
- 하단: 가이드 영역 + 진행률 + 다음 버튼

## 2. 입력 유형

### 2.1 단문 텍스트 입력
- 예: “당신을 무엇이라 불러드릴까요?”
- placeholder: “예: 김지수”
- 최소 1자, 최대 10자

### 2.2 숫자 입력 + 스텝퍼
- 예: “현재 몇 번째 생일을 지나오셨나요?”
- 스텝퍼(▲/▼) 제공
- 범위: 18~60(기본), 필요 시 확장 가능

### 2.3 단일 선택(카드/칩)
- 예: “당신이 가장 중요하게 생각하는 가치는 무엇인가요?”
- 2~4개 선택지 권장
- 선택 시 강조 테두리 + 체크

### 2.4 다중 선택(칩)
- 예: “우선순위는?”
- 최대 2~3개까지 선택 가능
- 선택 개수 가이드 표기

### 2.5 슬라이더
- 예: “월 상환액 허용 범위”
- 범위: 0~100(단위 커스텀)
- 실시간 값 표시

### 2.6 서술형(선택)
- 예: “꿈을 가로막는 장벽은?”
- 1~2줄 입력, 선택 질문으로 표기

## 3. 진행/피드백
- 상단 진행률: 5단계 중 현재 단계 표시
- 하단 진행률: 전체 질문 대비 현재 위치 표시
- 다음 버튼은 입력 완료 시 활성화
- 오류 메시지는 짧고 공감형 문장

## 4. 하단 가이드 영역
- 하단 좌측 고정
- 가이드 미니 프로필 + 한 줄 설명
- 길이: 1~2줄, 40자 내외
- 질문 변경 시 문구도 교체 가능

### 공통 문구(베이스)
- “지금 답변은 다음 단계의 지도를 그리는 데 쓰입니다.”
- “걱정하지 않아도 됩니다. 정답보다 방향이 중요합니다.”

### 카테고리별 문구 세트

#### 부동산 - 에이전트 한
- “입지와 시점, 이 두 가지가 집의 현실을 결정합니다.”
- “무리하지 않는 선에서 최적의 길을 찾겠습니다.”
- “지금 답이 없어도 괜찮습니다. 후보를 좁히는 중입니다.”

#### 기초자산(보험·연금) - 가디언 송
- “방어막이 단단할수록 선택은 자유로워집니다.”
- “급할수록 기본을 점검하는 게 가장 빠릅니다.”
- “안전장치를 먼저 세우고, 그 다음 성장을 보겠습니다.”

#### 주식 - 트레이더 최
- “흔들림을 줄이는 건 원칙과 기간입니다.”
- “속도보다 방향이 중요합니다. 흐름을 함께 보죠.”
- “작은 규칙 하나가 큰 손실을 막아줍니다.”

#### 라이프밸런스 - 디렉터 유
- “목표가 선명해질수록 돈의 쓰임도 선명해집니다.”
- “당신의 삶을 먼저 그리면, 숫자는 따라옵니다.”
- “지금의 선택이 더 나다운 미래를 만듭니다.”

## 5. 가이드 연출
- 질문 시작 전 1줄 인사 카드
- 질문 중간에 짧은 코멘트(선택)
- 마지막 질문 후 “최종 분석 시작” 문구

## 6. 접근성/모바일
- 터치 영역 최소 44px
- 키보드 입력 시 자동 스크롤
- 모바일에서 Next 버튼 고정
- 가이드 영역은 줄바꿈 허용

## 7. 카테고리별 입력 타입 예시
- 부동산: 단일 선택 + 슬라이더
- 기초자산: 단일 선택 + 다중 선택
- 주식: 단일 선택 + 슬라이더
- 라이프밸런스: 단일 선택 + 서술형

## 8. 톤 가이드
- 존댓말 유지
- 불안 자극 대신 “선택을 돕는” 문장 사용

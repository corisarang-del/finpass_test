# 상세 흐름 - 부동산 코스 (에이전트 한)

## 1. 시작 상황
- “내 집 마련”에 대한 막연한 불안을 정리하는 흐름

## 2. 주요 선택지 예시
- 주거 형태: 전세 / 매매 / 아직 모르겠음
- 희망 시점: 3년 내 / 5년 내 / 7년 이상
- 목표 지역 성향: 도심 / 근교 / 유연

## 3. 카테고리 추가 질문 예시
- 희망 입지의 우선순위는 무엇인가요? (직주근접, 학군, 자연환경)
- 현실적으로 감당 가능한 월 상환액은 어느 정도인가요?
- 전세에서 매매로 전환할 의향은 어느 정도인가요?

## 4. 가이드 등장
- “이번 코스를 담당하게 된 수석 가이드 에이전트 한입니다.”

## 5. 가이드 피드백(예시 대사)
- “현재 자산 구조로는 도심 매매가 오버페이스입니다. 청약 기회를 노리는 전략이 더 현실적입니다.”

## 6. 최종 분석 포인트
- 주거 목표 시점의 현실성
- 대출 리스크와 대안 경로(청약/전세 전략)

## 7. 결과 요약
- 현재 단계 진단 + 주거 계획 로드맵 초안

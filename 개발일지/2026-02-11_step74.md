# 개발일지 - 2026-02-11 (PS 로고 제거 - img → background-image)

작성시각: 2026-02-11 18:09

## 해결하고자 한 문제
- 마우스를 움직이면 이미지 위에 Adobe Photoshop(PS) 로고가 나타남

## 해결된 것

### 원인 분석
- Adobe Creative Cloud 브라우저 확장 프로그램이 `<img>` 태그를 감지
- 이미지에 마우스를 올리면 "Edit in Photoshop" 오버레이(PS 로고) 표시
- 해결: `<img>` 태그 → CSS `background-image`로 변경

### Home.tsx 수정

#### 1. 인트로 배경 이미지 (54-68줄)
```tsx
// 변경 전
<img src={bgImage} alt="Background" className="w-full h-full object-cover opacity-60" />

// 변경 후
<motion.div
  style={{
    backgroundImage: `url(${bgImage})`,
    backgroundSize: 'cover',
    backgroundPosition: 'center',
    opacity: 0.6,
  }}
>
```

#### 2. 카테고리 배경 이미지 (140-155줄)
```tsx
// 변경 전
<img src={cat.image} style={{ objectFit: 'cover', ... }} />

// 변경 후
<div
  style={{
    backgroundImage: `url(${cat.image})`,
    backgroundSize: 'cover',
    backgroundPosition: 'center',
  }}
/>
```

## 기술적 세부사항

### img 태그 vs background-image
- `<img>` 태그: DOM 요소로 존재 → 브라우저 확장 프로그램이 감지 가능
- `background-image`: CSS 속성 → 브라우저 확장 프로그램이 감지 불가
- 시각적으로는 동일한 결과

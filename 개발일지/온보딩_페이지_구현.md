# 온보딩 페이지 구현 개발일지

**작성시각**: 2026-02-13 00:56
**해결하고자 한 문제**: PINPASS 참조 이미지 기반 3단계 온보딩 플로우를 카테고리 선택 전에 추가

## 해결된 것

- `Onboarding.tsx` 신규 생성 (3단계: 가치 제안 → 공감 → 꿈 상상)
- `App.tsx`에 `/onboarding` 라우트 추가
- `Home.tsx` 수정: 인트로 종료 → `/onboarding` 이동, 온보딩 완료 → 카테고리 표시
- Wizard 질문 화면과 동일한 UXUI 적용:
  - 흰 배경 + 미세 radial-gradient 장식
  - 원형 뒤로가기 버튼 (border #e2e5ea, shadow)
  - Pill 스텝 카운터 (1/3, 2/3, 3/3)
  - 3px 프로그레스 바 (accent color 그라데이션)
  - 카드 스타일: border #e5e7eb, borderRadius 14, padding 14-18px
  - CTA 버튼: accent color gradient, borderRadius 12
  - lucide-react 아이콘 사용 (이모티콘 대신)
- 한 화면에 모든 콘텐츠 표시 (스크롤 없음)

## 안된 것

- 1차: 흰 배경 + 이모티콘 스타일로 만들었는데 홈과 안어울림 피드백
- 2차: 다크 시네마틱으로 했는데 Wizard 스타일이 맞다는 피드백
- 3차: Wizard 동일 스타일로 최종 수정 완료

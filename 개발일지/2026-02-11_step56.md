# 개발일지 - 2026-02-11 (Home Screen Scroll Redesign)

작성시각: 2026-02-11 17:15
해결하고자 한 문제:
- 카테고리 카드 렌더링 오류 (높이 0 붕괴).
- 사용자가 원하는 "스크롤 시 영상/이미지 확대" 인터랙션 부재.

변경 사항:
- `Home.tsx` 아키텍처 변경: `Timer Sequence` -> `Scroll Sequence`.
- **Scroll Container**: 전체 페이지 높이를 `300vh` 정도로 설정하여 스크롤 공간 확보.
- **Sticky Hero**: `position: sticky`와 `top: 0`을 활용하여 초기 화면 고정.
- **Animation**:
    - `scrollYProgress` [0, 0.3]: 이미지 컨테이너가 `80% width` -> `100% width`로 확장.
    - `scrollYProgress` [0.3, 0.6]: 메인 텍스트 Fade Out, 카테고리 섹션 등장 준비.
    - `scrollYProgress` [0.6, 1.0]: 카테고리 그리드 스크롤 업 & 안착.
- **Resource**: 배경 이미지를 영상 느낌의 고화질 이미지(Unsplash)로 고정. 

이점:
- 카드 높이가 찌그러지는 Flexbox 버그 원천 차단 (일반 Document Flow 사용).
- 최신 트렌드(Basedesign 스타일) 반영으로 사용자 만족도 제고.

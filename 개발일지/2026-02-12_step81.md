# Step 81 - Wizard 뒤로가기/입력방식/대화창 수정

## 작성시각
2026-02-12 19:09

## 해결하고자 한 문제
1. 왼쪽 상단 뒤로가기 버튼이 홈으로 이동 → 이전 단계로 이동해야 함
2. 직접 입력하는 질문이 너무 많음 → 이름/나이만 텍스트, 나머지 메뉴 버튼으로 변경
3. 하단 가이드 대화창 높이가 상태에 따라 바뀜 → 고정

## 해결된 것
- **뒤로가기**: `navigate('/')` → `currentStepIndex > 0`이면 이전 단계로, 첫 단계면 홈으로
- **텍스트→메뉴 변환**:
  - i4 (갑작스러운 지출) → single: 생활비/보험료/저축투자/대출상환
  - s2 (변동성 허용) → single: 매우 낮음/낮음/보통/높음 (slider→single)
  - s4 (월 투자 여력) → single: 10만원 이하/10~30/30~50/50이상 (slider→single)
  - s5 (투자 테마) → single: AI반도체/바이오헬스/2차전지/배당주
  - r3 (월 상환액) → single: 50만원 이하/50~100/100~150/150이상 (slider→single)
  - l1 (삶의 목표) → single: 내 집 마련/경제적 자유/건강한 삶/커리어 성장
  - l2 (목표 시점) → single: 1년/3년/5년/10년 이내
  - l3 (줄일 지출) → single: 외식카페/구독서비스/쇼핑의류/없음
  - l5 (하고 싶은 일) → single: 여행/가족과 시간/취미 몰입/새로운 도전
  - l6 (방해 습관) → single: 충동소비/미루기/과도한 야근/무계획
- **대화창 높이**: showChoices 조건부 높이 → 고정 `minHeight: 120`

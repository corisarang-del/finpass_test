# 개발일지 - 2026-02-11 (Home Layout Finalization)

작성시각: 2026-02-11 18:45
해결하고자 한 문제:
- 중앙 텍스트가 왼쪽 상단에 박히는 현상 (Centering Fail).
- 카테고리 이미지가 화면을 다 채우지 못하고 짤막하게 나옴 (Height Fail).

변경 사항:
- `Home.tsx`:
    - **Global Layout Strategy**: `relative` 의존성을 버리고 중요 요소(텍스트, 최종 메뉴)를 `fixed`로 뷰포트에 직접 배치.
    - **Hero Text**: `fixed inset-0 z-40 pointer-events-none flex items-center justify-center`.
    - **Category Menu**:
        - `fixed inset-0 z-50 flex flex-col md:flex-row`.
        - Navbar보다 위에 오도록(`z-50`) 하거나, Navbar를 포함하는 디자인으로 변경. (여기서는 Navbar는 Logo만 남기고 뒤로 보내거나 생략하여 Full Screen 몰입감 유도).
    - **Text Align**: 텍스트가 절대 겹치지 않도록 `group-hover` 시 `translate` 대신 `grid`나 `flex` gap을 활용한 자연스러운 밀어내기 적용 고려했으나, 현재는 `translate` 값을 안전하게 조정.

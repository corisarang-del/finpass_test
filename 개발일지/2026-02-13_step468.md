# Step 468 개발일지

- 작성시각: 2026-02-13 18:57:14
- 해결하고자 한 문제:
  - Vercel에서 `404: NOT_FOUND`가 반복되는 문제를 Root Directory 설정 의존성 없이 완화.

- 해결된 것:
  - 저장소 루트에 `vercel.json` 추가.
  - 설정값:
    - installCommand: `pnpm install --dir finpass-web`
    - buildCommand: `pnpm --dir finpass-web build`
    - outputDirectory: `finpass-web/dist`
    - rewrite: `/(.*)` -> `/index.html`
  - 루트 배포 기준으로도 `finpass-web` 앱이 빌드/서빙되도록 강제.

- 해결되지 않은 것:
  - Vercel 대시보드에서 프로젝트가 잘못된 저장소를 바라보는 경우는 별도 수정 필요.
  - 재배포(Redeploy) 전까지는 기존 404가 유지될 수 있음.

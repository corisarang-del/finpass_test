# 개발일지 - 2026-02-11 (텍스트 가시성 강화 - 글로우 효과)

작성시각: 2026-02-11 18:25

## 해결하고자 한 문제
- 어두운 배경 이미지(지구 야경 등)에서 FINPASS 로고와 인트로 텍스트가 보이지 않음

## 해결된 것

### 핵심 원인
- 이전에 어두운 그림자(dark shadow)를 사용했는데, 배경도 어두워서 글자가 묻힘
- 해결: 밝은 글로우(white glow) + 반투명 배경 패널 추가

### 1. FINPASS 로고 (73-82줄)
- 밝은 글로우 효과: `textShadow: '0 0 10px rgba(255,255,255,0.8), 0 0 20px rgba(255,255,255,0.5)'`
- 인라인 `color: '#ffffff'`로 확실한 흰색

### 2. 인트로 텍스트 (88-103줄)
- 반투명 검은 배경 패널: `backgroundColor: 'rgba(0,0,0,0.35)'`
- 배경 블러: `backdropFilter: 'blur(4px)'`
- 텍스트 밝은 글로우: `textShadow: '0 0 15px rgba(255,255,255,0.7)'`
- 인라인 `color: '#ffffff'`

- 디자인 퀄리티 개선 요청 ("촌스럽다" -> "레퍼런스 사이트 느낌으로")

원인 분석:
- 스크린샷 상 가이드 이미지가 `w-12` 클래스를 무시하고 원본 비율이나 큰 사이즈로 렌더링 된 것으로 추정.
- Tailwind CSS가 로드되지 않았거나, `img` 태그의 기본 스타일 간섭 가능성.
- `fixed` 포지셔닝이 모바일/데스크탑 환경 차이로 인해 겹침 발생.

해결 계획:
- **`Wizard.tsx`**:
    - 가이드 위치를 하단 고정에서 **상단 헤더 아래** 또는 **질문 위**로 이동하여 겹침 원천 차단.
    - 이미지에 `style={{ width: 50, height: 50 }}` 등 인라인 스타일을 병행하여 CSS 로드 실패 시에도 레이아웃 방어.
    - 전체 레이아웃을 `flex-col`로 잡고 `flex-1`을 메인에 할당, 스크롤 가능하도록 변경.
- **`Home.tsx`**:
    - 레퍼런스(FinPass)의 깔끔한 화이트 톤, 큰 타이포그래피, '시작하기' 버튼 강조 형태로 변경.

# 개발일지 - 2026-02-11 (Home Text & Layout Alignment Fix)

작성시각: 2026-02-11 18:30
해결하고자 한 문제:
- Hero Text가 중앙이 아닌 좌측 상단으로 이동하여 로고와 겹치는 심각한 버그.
- 카테고리 텍스트가 서로 겹침.
- 카테고리 하나가 화면에서 사라짐.

변경 사항:
- `Home.tsx`:
    - **Hero Section**: `absolute` 컨테이너에 `z-30` 및 `pointer-events-none`을 추가하고, Flex Centering을 더욱 견고하게 작성.
    - **Category Grid**: `flex` 기반 분할에서 `grid` 기반 분할(`grid-cols-1 grid-rows-4 md:grid-cols-4 md:grid-rows-1`)로 변경하여 화면 꽉 채움 보장.
    - **Typography**: 
        - 카드 내부 텍스트를 `absolute`가 아닌 `static`으로 배치.
        - `space-y-2` 등을 활용하여 제목과 설명 사이의 물리적 간격 확보.
        - 줄바꿈 방지(`whitespace-nowrap`) 유지.

## 수행 내용
1. `ValueDiscovery` 개편
   - 폰트/카드 레이아웃 정리
   - 핵심목표/가치 기본값을 기존 입력값(`answers`) 기반으로 자동 선택
2. `VisionBoard` 시각화 강화
   - 비전 항로 그래픽(점/라인) 추가
   - 카드형 비전 정보 가독성 개선
3. `LiveSimulation` 개편
   - 카테고리 가이드 제거, `플래너 킴 + AI 코치` 고정
   - 슬라이더 조정량/지배 항목 기반으로 멘트 실시간 변경
4. `FinalAnalysis` 재구성
   - `미래 자산 등고선` 스타일 섹션 적용
   - 단계별(1~5) 자산/증가액 리스트 구성
5. 기획 문서 업데이트
   - 온보딩 입력 자동 반영 규칙
   - 플래너킴+AI 가이드 정책
   - 슬라이더 연동 멘트 변경 규칙
   - 미래 자산 등고선 UI 반영

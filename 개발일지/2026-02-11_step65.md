# 개발일지 - 2026-02-11 (홈화면 및 카테고리 레이아웃 수정)

작성시각: 2026-02-11 16:58

## 해결하고자 한 문제
1. 홈화면 인트로 텍스트가 왼쪽 상단으로 치우쳐서 FINPASS 로고와 겹침
2. 카테고리 선택화면에서 이미지가 절반만 보이고 큰 여백 발생
3. 카테고리 호버 효과 개선 필요

## 해결된 것
### 1. 인트로 텍스트 중앙 정렬 강화
- `fixed inset-0` 컨테이너에 `style={{ top: 0, left: 0, right: 0, bottom: 0 }}` 명시적 스타일 추가
- motion.div에 `justify-center` 및 `mx-auto` 추가로 완벽한 중앙 정렬
- `text-center` 클래스를 span과 h1 모두에 추가
- `mix-blend-screen` 제거하고 `drop-shadow-2xl`로 가독성 개선

### 2. 카테고리 이미지 풀사이즈 표시
- 모바일에서 `min-h-[200px]` 추가하여 최소 높이 확보
- 이미지 오버레이를 단순 `bg-black/50`에서 `bg-gradient-to-b` 그라데이션으로 변경하여 텍스트 가독성 향상
- `object-cover` 유지하여 이미지가 전체 영역을 채우도록 보장

### 3. 호버 효과 개선
- 이미지 확대 효과를 `scale-110`에서 `scale-105`로 줄여 자연스럽게 개선
- 애니메이션 duration을 700ms에서 500ms로 단축하여 반응성 향상
- 배지(Badge) 효과 개선: 항상 표시되도록 하되 호버 시 opacity 및 border 변경
- Description을 하단 절대 위치에서 제거하고 카드 내부에 통합하여 겹침 방지
- 호버 시 컨텐츠가 살짝 위로 이동하며 확대되는 효과 추가 (`-translate-y-2 scale-105`)
- Description에 `delay-100` 추가하여 단계적 애니메이션 구현

### 4. 기타 개선
- `ArrowRight` import 제거 (사용하지 않음)
- 카테고리 타이틀 크기 증가: `text-3xl md:text-5xl`에서 `text-4xl md:text-5xl lg:text-6xl`로 변경
- 패딩 조정: 모바일 `p-6`, 데스크탑 `p-8`로 반응형 개선

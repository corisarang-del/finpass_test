# Step 85 - 선택 버튼 클릭 시 다크 스타일 피드백 추가

## 작성시각
2026-02-12 19:54

## 해결하고자 한 문제
- 선택 버튼 클릭 시 다크네이비 배경으로 바뀌는 효과가 보이지 않음
- `handleOptionSelect`가 즉시 다음 스텝으로 넘어가서 선택 상태 확인 불가

## 해결된 것
- `handleOptionSelect` 함수에 `setTimeout(500ms)` 딜레이 추가
- 선택 즉시 `setInputValue`로 상태 변경 → 다크네이비 카드 렌더링
- 0.5초 후 다음 스텝으로 넘어감
- 사용자가 선택 피드백(다크 배경 + 체크 아이콘)을 확실히 확인 가능

## 해결 안된 것
- 없음

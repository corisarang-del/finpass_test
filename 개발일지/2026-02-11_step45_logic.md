# 개발일지 - 2026-02-11 (Wizard 로직 개선)

작성시각: 2026-02-11 16:05
해결하고자 한 문제:
- `Wizard.tsx`에 조건부 질문(`condition`) 속성을 추가했으나, 실제 렌더링 로직(`useMemo`)에는 반영되지 않아 모든 질문이 노출되는 문제 예상.
- 조건부 질문이 스킵될 때 인덱스 처리 및 `handleNext` 로직의 유연성 확보 필요.

변경 사항:
- `questions` useMemo 수정:
    - `answers` 상태가 변경될 때마다 질문 리스트를 다시 필터링하도록 수정.
    - `condition(answers)`가 `true`인 질문만 리스트에 포함.
- `handleNext`, `handleOptionSelect` 수정:
    - 조건부 로직에 의해 질문 리스트 길이가 변하더라도 `isLastStep` 판별이 정확히 동작하도록 수정.
    - 답변 저장 시 `newAnswers`를 생성하여 즉시 `navigate`나 상태 업데이트에 반영.

기대 효과:
- "싱글" 선택 시 파트너 관련 질문이 자동으로 숨겨짐.
- 기획 의도에 맞는 맞춤형 질문 흐름 제공.

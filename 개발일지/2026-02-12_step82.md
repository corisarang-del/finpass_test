# Step 82 - Wizard 가이드 자기소개 단계 추가

## 작성시각
2026-02-12 19:15

## 해결하고자 한 문제
- 가이드가 자기소개를 먼저 하고, 그 후에 이름을 입력받도록 순서 수정

## 해결된 것
- `intro` 타입 새로 추가 (QuestionType에 'intro' 추가)
- 공통 질문 맨 앞에 `c0` (가이드 자기소개 단계) 추가
- intro 단계에서 가이드 자기소개(`guide.intro`)를 타이핑 효과로 표시
- 타이핑 완료 후 "시작하기" 버튼 표시 → 클릭하면 다음 단계(이름 입력)로 이동
- intro일 때 텍스트/숫자 입력 UI 숨김 처리

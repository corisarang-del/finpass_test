# Step 83 - Wizard 배경/폰트 프리미엄 스타일 적용

## 작성시각
2026-02-12 19:28

## 해결하고자 한 문제
- 가이드 선택화면 배경이 회색 그라데이션으로 세련되지 않음
- 폰트가 고급 컨설팅 느낌이 아님
- 선택지 스타일이 레퍼런스 이미지와 다름

## 해결된 것
- **배경**: 회색 그라데이션(`#f8f9fb→#e8ecf3`) → **순수 화이트**(`#ffffff`)
- **질문 헤딩**: 선택지 위에 fontSize 22, fontWeight 700, letterSpacing -0.02em으로 크고 세련되게 표시
- **선택지 카드**: 둥근 카드(borderRadius:14, 그림자) → **깔끔한 리스트 스타일**(borderRadius:0, 좌측 컬러 바, 하단 구분선)
- **선택 시**: 가이드 컬러 좌측 바(borderLeft 3px) + 연한 배경 틴트
- **호버**: 미세한 배경 변화 + 좌측 바 표시
- **multi 선택지**: 동일한 리스트 스타일로 통일
- **색상**: `#1a1a2e` → `#2c2c3a`로 좀 더 부드러운 다크

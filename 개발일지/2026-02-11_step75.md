# 개발일지 - 2026-02-11 (밝기 및 뱃지 위치 수정)

작성시각: 2026-02-11 18:14

## 해결하고자 한 문제
1. 인트로 화면이 너무 어두워서 FINPASS 로고가 안 보임
2. 카테고리 뱃지(현실점검, 안전확보 등) 글자가 약간 짤림

## 해결된 것

### 1. 인트로 배경 밝기 증가
- background opacity: 0.6 → 0.85
- 오버레이: bg-black/30 → bg-black/10

### 2. FINPASS 로고 가시성 강화
- text-white/90 → text-white
- drop-shadow-md → textShadow 인라인 스타일 (더 강한 그림자)

### 3. 카테고리 뱃지 잘림 해결
- 컨텐츠 패딩 추가 (padding: 8px 16px)
- 뱃지 글자 크기 축소: text-[10px] → text-[9px]
- 뱃지 패딩 축소: px-3 py-1.5 → px-2 py-1
- 타이틀 크기 축소: text-4xl → text-2xl (모바일)
- 요소 간 간격 축소: space-y-3 → space-y-1
- 설명 텍스트 크기 축소: text-sm → text-xs

# Step 78 - Wizard 레이아웃 수정 (캐릭터 왼쪽 + 선택지 오른쪽 중앙)

## 작성시각
2026-02-12 18:43

## 해결하고자 한 문제
1. 선택지 메뉴가 화면에서 잘려서 전부 안 보이는 문제
2. 선택지가 캐릭터 얼굴을 가리는 문제
3. 가이드 캐릭터가 오른쪽이 아닌 왼쪽에 있어야 하는 요구사항

## 해결된 것
- 캐릭터 이미지를 오른쪽에서 왼쪽(`left: 2%`)으로 이동
- 선택지를 오른쪽 중앙(`right: 8%`, `width: 45%`)에 배치 → 캐릭터 안 가림
- 옵션이 많아서 화면에 안 들어가는 질문 축소:
  - c3 가치 질문: 6개 → 4개 (성장/안정/자유/건강)
  - r2 입지 우선순위: 5개 → 4개 (자연환경 제거)
  - r5 걱정 변수: 4개 → 3개 (집값하락 제거)
  - i1 비상금: 5개 → 4개 (없음/3개월미만/3~6개월/6개월이상)
- gap 12→10, padding 제거로 더 컴팩트하게 배치

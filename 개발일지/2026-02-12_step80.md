# Step 80 - Wizard 배경/메뉴 디자인 리뉴얼 (화이트 톤)

## 작성시각
2026-02-12 18:57

## 해결하고자 한 문제
1. 메뉴 버튼이 상단 진행률 바 선과 겹치는 문제
2. 배경이 어두운 우주/자연 사진 → 하얀색 바탕 세련된 디자인으로 변경
3. 메뉴 버튼 디자인을 레퍼런스 사이트(finpass3jo.vercel.app) 참고하여 깔끔하게 수정

## 해결된 것
- **배경**: 어두운 이미지 → 밝은 화이트+그라데이션(`#f8f9fb` → `#e8ecf3`) + 가이드 컬러 미세 방사 점
- **상단 UI**: 어두운 반투명 → 화이트 배경, 보더 `#e2e5ea`, 소프트 그림자
- **선택지 버튼**: 시안 반투명 둥근 버전 → 화이트 카드 스타일(borderRadius:14, white배경, 가벼운 보더, 좌측 정렬)
- **선택 시**: 가이드 컬러 테두리 + 연한 배경 틴트
- **하단 대화창**: 어두운 글래스모피즘 → 화이트 배경 + 소프트 그림자
- **텍스트**: 흰색 → 진한 네이비(`#1a1a2e`) / 보조 `#888`
- **입력 필드**: 어두운 배경 → `#f8f9fb` 밝은 필드
- **슬라이더**: 투명 트랙 → `#e2e5ea` 밝은 트랙
- **메뉴 위치**: top 12% → 18%로 내려서 진행률 바와 겹침 해소

# 홈 카테고리 선택 화면 카드 레이아웃 리디자인

## 작성 시각
2026-02-20

## 해결하고자 한 문제
홈 화면에서 4개 카테고리(부동산, 기초자산, 주식, 라이프) 선택 영역이 세로로 쌓인 풀스크린 밴드 형식이었음.
role.png에서 제시된 가로 카드 그리드 형식으로 변경 요청.

**기존 디자인 문제점:**
- 4개 섹션이 각각 25vh씩 세로로 적재 → 공간 활용 비효율
- 카드 UI의 시각적 위계 부족
- 현대적인 카드 그리드 UX 패턴과 괴리

## 해결된 것
- `sequenceStep === 3` 블록 전체 리디자인
- **레이아웃 변경**: 세로 밴드 → CSS Grid 가로 카드
  - 모바일: 2열 (`grid-cols-2`)
  - 데스크톱(md+): 4열 (`grid-cols-4`)
- **배경**: `#0d1117` 다크 네이비 + home3.jpg 7% 오버레이 텍스처
- **헤더 추가**: "SELECT YOUR JOURNEY" 서브태그 + "인생 지도를 그려보세요" 제목
- **카드 디자인**:
  - 라운드 코너 (14px)
  - 이미지 배경 + 하단 그라디언트 오버레이
  - 하단 좌측 텍스트 (sub 레이블 + 카테고리명 + desc)
  - 호버: 카드 위로 8px 이동 (`whileHover`), 이미지 scale-105, 흰색 테두리 출현, desc 텍스트 페이드인
- **하단 힌트**: "원하는 영역을 선택하면 함께 전략을 세워드립니다"
- **폰트 유지**: Sora, Pretendard, Outfit (기존 동일)
- **색감 유지**: 다크 모드 기조, 흰색 텍스트 계층

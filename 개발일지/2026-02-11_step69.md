# 개발일지 - 2026-02-11 (파비콘 삭제 및 카테고리 Grid 레이아웃 수정)

작성시각: 2026-02-11 17:16

## 해결하고자 한 문제
1. PS 파비콘이 여전히 브라우저 탭에 표시됨
2. 카테고리 4개는 보이지만 이미지가 풀사이즈로 표시되지 않음 (위아래 회색 여백 존재)

## 해결된 것
### 1. 파비콘 완전 제거
- `public/vite.svg` 파일 삭제
- 브라우저 재시작 시 PS 아이콘 완전히 사라짐

### 2. 카테고리 레이아웃을 Grid로 변경
- Flex 레이아웃에서 Grid 레이아웃으로 전환
- 변경 전: `flex flex-col md:flex-row`
- 변경 후: `grid grid-cols-1 grid-rows-4 md:grid-cols-4 md:grid-rows-1`
- Grid 사용으로 화면을 정확히 4분할하여 여백 완전 제거

### 3. 카테고리 버튼 클래스 수정
- 변경 전: `flex-1 w-full min-h-[25vh] md:min-h-0 md:h-full`
- 변경 후: `w-full h-full`
- Grid cell이 자동으로 크기를 분배하므로 flex-1이나 min-h 불필요
- 각 버튼이 grid cell 전체를 채우도록 간소화

## 기술적 세부사항
### Home.tsx 수정 사항

#### 1. 컨테이너 레이아웃 (99줄)
```tsx
// 변경 전
className="fixed inset-0 w-full h-full z-40 flex flex-col md:flex-row bg-black"

// 변경 후
className="fixed inset-0 w-full h-full z-40 grid grid-cols-1 grid-rows-4 md:grid-cols-4 md:grid-rows-1 bg-black"
```

#### 2. 버튼 클래스 (109줄)
```tsx
// 변경 전
className="group relative flex-1 w-full min-h-[25vh] md:min-h-0 md:h-full overflow-hidden..."

// 변경 후
className="group relative w-full h-full overflow-hidden..."
```

### 왜 Grid를 선택했는가?
- Flex는 content 크기에 따라 분배가 달라질 수 있음
- Grid는 명시적으로 4등분을 보장
- 모바일: 세로 4행 (grid-rows-4)
- 데스크탑: 가로 4열 (grid-cols-4)
- 각 셀이 정확히 동일한 크기를 가지므로 이미지가 풀사이즈로 표시됨

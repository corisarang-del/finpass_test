# Step 84 - Wizard 선택 버튼 카드 스타일 리디자인

## 작성시각
2026-02-12 19:36

## 해결하고자 한 문제
1. 4번째 선택 버튼이 하단 가이드 상태창과 겹침
2. 선택 버튼이 밋밋한 리스트 스타일 → 세련된 카드 스타일로
3. 클릭 시 선택됐다는 느낌이 약함 → 확실한 시각적 피드백

## 해결된 것
- **레이아웃**: 리스트 → **2x2 그리드 카드** (`gridTemplateColumns: repeat(2, 1fr)`)
- **선택지 위치**: 오른쪽 편향 → **화면 중앙**(`left: 50%`, `transform: translateX(-50%)`)
- **maxHeight**: 58vh → 52vh로 줄여서 하단 겹침 해소
- **카드 스타일**: 둥근 모서리(borderRadius:16), 미니멀 보더, 최소 높이 80px
- **선택 시**: **다크네이비(#1e2a3a) 배경** + 흰 텍스트 + 체크 아이콘 (오른쪽 상단)
- **그림자**: 선택 시 `0 8px 25px rgba(30,42,58,0.25)`, 미선택 `0 1px 3px`
- **호버**: 보더 색상 변화 + 살짝 올라감(`translateY(-2px)`)
- **질문 텍스트**: 중앙 정렬, fontSize 24, fontWeight 700
